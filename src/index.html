Projecten

Data engineering project 1

Dit project had als doel vluchtinformatie te scrapen van TUI, Ryanair, Transavia en Brussels Airlines. Het project is volledig opgebouwd in Python en is ontworpen om standalone te draaien op een Almalinux 9 server. De verzamelde gegevens worden opgeschoond en ingeladen in een MySQL database als OLTP database. In dezelfde MySQL-omgeving wordt een Datawarehouse opgezet. Door ee nverbinding te maken tussen PowerBI en MYSQL kunnen er meerdere analyserapporten gemaakt worden van de veramelde gegevens.
